StuddyBuddy

StuddyBuddy is a all-in-one web app designed to help you stay focused, organized, and just a little bit happier while studying. 

Features
- Pomodora Timer: stay focused with times study sessions 
- Countdown Timer: track deadlines or upcoming events
- To-Do List: write, complete, and manage tasks
- Tally Counter: count anything you want (habits, reps, etc)
- Notes/Journal: jot down thoughts or study notes
- Monkey Runner Game: a simple runner game for quick mental breaks 

Built With
- HTML
- CSS (custom styling and Google Fonts)
- Vanilla Javascript
- Canvas API (for the monkey runner game)

Getting Started
1. clone the repository 
2. open the index.html in your browser
